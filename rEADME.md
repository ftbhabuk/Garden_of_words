## ✨ About

Garden of Words is an elegant AI-powered poetry generation platform that transforms your ideas into sophisticated verses. Built with modern web technologies and advanced AI integration, it provides a serene creative space where every thought can bloom into beautiful poetry.

## 🚀 Features

- **🎭 Multiple Poetry Forms**: Haiku, Sonnet, Free Verse, Limerick, and more
- **🎨 Rich Styling Options**: Romantic, Contemporary, Nature, Urban, Philosophical themes
- **🎵 Tone Control**: Lyrical, Dramatic, Contemplative, Whimsical, Dark moods
- **🤖 Advanced AI Integration**: Powered by Groq and OpenAI with custom prompt engineering
- **💫 Beautiful UI/UX**: Glassmorphism design with smooth Framer Motion animations
- **⚡ Real-time Generation**: Instant poetry creation with intelligent suggestions
- **📱 Responsive Design**: Seamless experience across all devices

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS, Framer Motion, Radix UI
- **AI**: Groq, OpenAI, Custom prompt engineering
- **Deployment**: Vercel
- **Analytics**: Vercel Analytics & Speed Insights

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/garden-of-words.git
cd garden-of-words

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your API keys for Groq/OpenAI

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the magic ✨

## 🎯 How It Works

1. **Choose Your Inspiration**: Enter a thought, emotion, or scene
2. **Select Poetry Form**: Pick from Haiku, Sonnet, Free Verse, etc.
3. **Set the Mood**: Choose your preferred style and tone
4. **Watch Creation**: AI crafts your personalized verse in seconds
5. **Refine & Share**: Edit, regenerate, or share your creation
